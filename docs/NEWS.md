# News

This page highlights recent user-visible and maintenance milestones. For the
full dated record, see [CHANGELOG.md](CHANGELOG.md).

- Added Codex `PreToolUse` protocol support, including Codex decision JSON,
  nullable transcript paths, and empty output for passthrough results.

  example.

  for installs, remote execution, and other higher-impact actions.

  metadata commands.
- Clarified the usage guide's focus on policy principles and live deny reasons.
